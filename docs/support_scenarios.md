# gala-anteater 支持的故障诊断场景

## anteater 模型简介

* 当前 anteater 主要基于历史数据进行无监督训练，然后进行异常检测；
* 训练数据集应保证**无故障数据**，数据量不小于 14h；
* 推理阶段，按分钟级别进行推理，最小推理时间 1 分钟；
* 针对不同的场景，构建不同的模型，如应用级检测主要为“深度神经网络模型”、系统级为 “Rule-based” + “n-sigma” 模型，JVM OOM 主要为：Tree-based 模型。
